# Introducción a Next.js

Next.js es un poderoso framework de React que ha revolucionado la forma en que construimos aplicaciones web modernas. Ofrece una experiencia de desarrollo excepcional con características como renderizado del lado del servidor (SSR), generación de sitios estáticos (SSG) y enrutamiento basado en archivos.

## Características principales

1. **Renderizado híbrido**: Next.js permite elegir entre SSR, SSG o renderizado del lado del cliente según las necesidades de cada página.

2. **Enrutamiento automático**: Crea rutas basadas en la estructura de archivos en el directorio `pages`.

3. **Optimización de imágenes**: Incluye un componente `Image` que optimiza automáticamente las imágenes para mejorar el rendimiento.

4. **API Routes**: Permite crear fácilmente endpoints de API dentro de la misma aplicación.
